# ENTEC Informática 💻

Site institucional da ENTEC Informática, empresa especializada em manutenção e suporte técnico para computadores e notebooks.

Este projeto foi desenvolvido como prática de front-end, aplicando HTML, CSS e JavaScript com foco em responsividade e experiência do usuário.

---

 🚀 Demonstração

🔗 Acesse o site online:
https://MAYKON-SIAM.github.io/entec/

---

🖥️ Sobre o projeto

O site apresenta os serviços da ENTEC Informática, incluindo:

- Manutenção de computadores e notebooks
- Formatação e remoção de vírus
- Upgrades de hardware
- Suporte técnico especializado
- Atendimento via WhatsApp e Instagram

---

🎯 Objetivo

Criar uma presença digital moderna e responsiva para a empresa ENTEC, com foco em:

- Clareza na apresentação dos serviços
- Design moderno e tecnológico
- Fácil acesso ao contato
- Boa experiência em dispositivos móveis

---

 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript
- Normalize.css
- Google Fonts
- Google Maps Embed

---

 📱 Responsividade

O projeto foi desenvolvido para diferentes tamanhos de tela:

- 📱 Mobile
- 📟 Tablet
- 💻 Desktop

---

✨ Funcionalidades

- Layout responsivo
- Carrossel de imagens
- Seção de benefícios
- Links diretos para WhatsApp e Instagram
- Mapa integrado do Google Maps
- Animações e efeitos visuais em CSS
